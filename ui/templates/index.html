Recommendation System for an e-commerce firm

Objective:

  • The firm aimed to optimize user satisfaction and reduce returning customer's order time.

Strategy:

  • Collected and cleaned data on user purchases and product interactions.

Outcome:

Face liveness detection using AI for a ed-tech firm

Objective:

  • Detect if students from the online classes are real and paying attention detecting eye blink frequency.

Strategy:

  • Used DeepFace, a deep learning-based facial recognition model, for robust face detection and landmark localization.
  • Implemented an algorithm to analyze blinking patterns by tracking changes in eye landmarks over consecutive frames.
  • Computed a liveness score based on the frequency and consistency of blinking within a defined time window.

Outcome:

  • Provided an additional layer of security by verifying the authenticity of faces, mitigating spoofing attempts using static images, generative AI or videos.
